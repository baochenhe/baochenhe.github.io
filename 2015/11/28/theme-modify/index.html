hexo yilia 主题多说评论

在小段果果帮助下接触到了hexo,跟着教学文档一直下来，也默认使用了推荐的yilai主题
Hexo主题Yilia下载
这款主题简单精美，也是我所喜欢的，期间也尝试到知乎上面去搜索，详情请见：

hexo有哪些好看的主题相关

感兴趣的朋友可以换来看看哟。

主要目的

本文的主要目的有三，记录和吐槽加反省…,虽然我很是感谢这款主题的作者
Litten 。 不过他留的这个坑也着实让我吃不消。

看到中间的关于多说的评论的用法

填写你在多说申请的项目名称 duoshuo: duoshuo-key
是的，于是我毫不犹豫的去多说注册账号填写各种得到了这个

多说项目设置

被主题作者带入了KEY的概念，可是我翻遍了这个后台都就发现 密钥和名称有可能会是我需要的，可是config.yml中写

duoshuo: true

true 对应过来的 应该是。。。。 false 陷入了深深地沉思。

一直看到评论框下面的版权：

对于有严重强迫症的我，真的非常的吐血，每秒1000点魔法伤害直至死亡节奏

hexo\themes\yilia\layout_partial\post\duoshuo.ejs
都证实了我的假设，内心千万头草泥马奔腾而过，也是我没够细心，是我菜。。。
